Removing Pandoc’s bibliography

Oct 22, 2019 • meta • pandoc, csl

In generating this website I use a combination of Jekyll1 and Pandoc.2 Pandoc converts Markdown files into HTML; I use it because it supports Citation Style Language (CSL),3 which Jekyll’s default Markdown processors do not.

I am generally of the view that footnotes are the most civilised way to cite others, though my justifications for this mainly apply to books. However, I am a creature of habit, and so like to have full footnotes in lieu of a bibliography when writing on-line too. However, Pandoc seemed to generate both a bibliography and full footnotes, which looked fairly bizarre.4

It turns out that CSL is not as alien as I thought; all that one needs to do to fix this is to remove everything between the <bibliography> tags on one’s local copy of the CSL file to cause the bibliography to disappear.
